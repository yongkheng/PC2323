Sustainable Cities Index Service API

API Endpoints

  • GET /api/sustain/{city}?factor={factor}

    Description: The api retrieves the Substainable Cities Index of a city based on social, environmental, and economic factors. There are 100 major global cities involved in this ranking. More information can be found at Arcadis
    Parameters
    • Required: city={city} :: specify the city to retrieve the sustainable index.
    • Optional: factor={factor} :: {factor} is one of the options: overall, people, planet, and profit
    Response JSON: Click for example output
    • /api/sustain/zurich
    • /api/sustain/penang
    • /api/sustain/london?factor=people
